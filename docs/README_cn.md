## 简介

EasyQQ 是一个增加腾讯QQ易用性的工具，其基于 [NoneBot](https://github.com/nonebot/nonebot) 框架和 [OneBot 标准](https://github.com/howmanybots/onebot)（原 CQHTTP） ，可以方便的帮助用户通过聊天消息的方式调用各种功能，并提升聊天的趣味性。

## 功能

- 对于普通用户
  - 统计投票功能
  - 禁言自己
  - 签到
  - 提问

- 对于群聊管理员（附加功能）
  - 禁用群消息
  - 禁言特定用户

- 趣味性功能
  - 自动“复读”
- 易用性功能
  - 敏感词检测（自动警告）
  - 运势查询
